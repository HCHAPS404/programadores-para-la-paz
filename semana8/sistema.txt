Que muestra cada comando:
El comando uname -a muestra informacion general del sistema, como el nombre del sistema operativo, el kernel, la arquitectura y otros datos tecnicos.
El comando whoami muestra el nombre del usuario que esta usando la terminal en ese momento.

Por que es util conocer esta informacion:
Conocer estos datos ayuda cuando se administra un servidor o se revisa un equipo, porque permite saber en que sistema se esta trabajando.
Tambien sirve para diagnosticar problemas, instalar herramientas compatibles y confirmar si se estan usando los permisos correctos.
