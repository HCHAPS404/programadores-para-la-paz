Prueba de reportes

POST http://localhost:3000/reportes
JSON:
{
  "tipo": "Infraestructura",
  "descripcion": "Dano en alumbrado publico"
}

Respuesta:
{
  "mensaje": "Reporte registrado",
  "reporte": {
    "id": 1,
    "tipo": "Infraestructura",
    "descripcion": "Dano en alumbrado publico"
  }
}

GET http://localhost:3000/reportes
Devuelve la lista de reportes guardados en memoria.
