Las APIs ayudan a que la información no quede regada en chats o mensajes sueltos.
Con una ruta POST se puede recibir un reporte con datos claros: tipo, mensaje, descripción, etc.
Después otra ruta GET puede mostrar lo que ya llegó.
Eso facilita organizar alertas, solicitudes o novedades de una comunidad.
También permite que una misma API la use una página web u otra aplicación más adelante.
En la práctica, una API es como un punto de entrada para enviar y consultar datos.
Para procesos participativos, tener esa estructura hace más fácil dar seguimiento a lo que reportan las personas.
