Prueba de la ruta POST /registro

URL: http://localhost:3000/registro

JSON enviado:
{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Respuesta:
{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor lee los datos desde req.body y devuelve el mismo contenido con un mensaje de confirmación.
